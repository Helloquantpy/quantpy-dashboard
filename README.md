# Quantpy — Forex Analysis Dashboard

Dashboard d'analyse Forex quantitative construit avec Python et Dash.

## Fonctionnalités
- Analyse technique (MM20, MM50, RSI)
- Simulation Monte Carlo (jusqu'à 1000 scénarios)
- Risk Manager (SL, TP, Ratio R/R)
- Export PDF professionnel

## Demo en ligne
https://forexdashboard.up.railway.app

## Technologies
Python | Dash | Plotly | yfinance | NumPy | SciPy
